The `git push` command is used to upload local repository content to a remote repository.

## Why Use It

Once you have committed your changes locally, you need to send them to your team's shared repository (like GitHub) so others can see them.

## Syntax

```bash
git push <remote> <branch>
```

## Examples

**Push to origin**
```bash
git push origin main
```

**Force push (USE WITH CAUTION)**
```bash
git push origin main --force
```

> Force pushing overwrites the remote history. Never use this on a shared branch!
