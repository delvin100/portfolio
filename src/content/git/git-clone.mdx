The `git clone` command is used to target an existing repository and create a clone, or copy of the target repository.

## Why Use It

Whenever you want to contribute to an existing project (like on GitHub or GitLab), you need a local copy on your machine. `git clone` downloads the entire history and all branches of the project.

## Syntax

```bash
git clone <url> [directory]
```

## Examples

**Clone a GitHub repository**
```bash
git clone https://github.com/user/repo.git
```

**Clone into a specific folder**
```bash
git clone https://github.com/user/repo.git my-folder
```
