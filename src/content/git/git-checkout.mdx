The `git checkout` command operates upon three distinct entities: files, commits, and branches. 

> In modern Git (>= 2.23), `git checkout` has been split into two safer commands: `git switch` (for changing branches) and `git restore` (for reverting files).

## Examples

**Switch branches**
```bash
git checkout feature-branch
```

**Create and switch to a new branch**
```bash
git checkout -b new-feature
```

**Discard changes in a file**
```bash
git checkout -- index.html
```
